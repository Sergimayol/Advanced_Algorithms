\begin{abstract}
\hspace{2pt}Esta aplicación presenta una interfaz gráfica de usuario que le permite interactivamente solucinar el \say{Knight's tour} para un conjunto de piezas genéricas (Reina, Caballo, Torre, etc) podiendo escoger la pieza que guste y su posición inicial. Sin embargo, se ha modificado el algoritmo para que el recorrido pueda ser compartido por el conjunto piezas presentes en el tablero, por lo que se irán turando por el orden de inserción al tablero. 
\end{abstract}
