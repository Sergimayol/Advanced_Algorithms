\subsection{Better swing}

Better swing\cite{betterswing} es una librería derivada de Java Swing que permite un desarrollo más amigable y sencillo de interfaces gráficas, al estar inspirado en el desarrollo web (HTML y CSS), concretamente, en el framework de \say{Bootstrap}. Se basa en \say{JfreeChart} para pintar las gráficas de líneas creando \say{wrappers} para facilitar su uso y manejabilidad.
