\begin{abstract}
\hspace{2pt}Esta aplicación presenta una gráfica de los costes computacionales asintóticos de tres algoritmos variando el tamaño \textit{N} del vector de datos. Los dos primeros encontrarán la moda con un coste asintótico entre $O(n)$ y $O(n\log{}n)$. El tercero implementará el producto vectorial del mismo vector por él mismo, representando así un coste $O(n^2)$.
\end{abstract}
