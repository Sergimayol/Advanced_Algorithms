\subsection{Better swing}

Better swing es una librería de derivada de java swing que permite un desarrollo más amigable y sencillo de interfaces gráficas, al estar inspirado en el desarrollo web (HTML y CSS), concretamente, en el framework de \say{Bootstrap}. Se basa en \say{JfreeChart} para pintar las gráficas de líneas creando \say{wrappers} para facilitar su uso y manejabilidad.
